## Description

`app.py` dan `main.py` menggunakan `import requests` tetapi `requests` tidak ada di `requirements.txt`. Fresh install dengan `pip install -r requirements.txt` akan gagal saat menjalankan frontend.

**Konfirmasi dari `progress_log.md` developer:**
> *"`requests` belum ada di `requirements.txt` (penyebab: `app.py`/frontend `import requests`) — perlu ditambahkan agar Streamlit UI bisa berjalan setelah `pip install -r requirements.txt`."*

Developer telah mengakui issue ini secara eksplisit di progress log namun belum di-fix.

**Kode aktual `requirements.txt`:**
```
groq
python-dotenv
streamlit
sentence-transformers
numpy
pypdf
fastapi
uvicorn[standard]
sqlalchemy
python-multipart
# ← requests TIDAK ADA
```

**Re-test Status:** ❌ **FAIL — Bug Still Open**

---

## Recommendation

Tambahkan satu baris ke `requirements.txt`:
```
requests
```

**Workaround sementara:**
```bash
pip install requests
```
