## Description

Sidebar menampilkan `help="Maksimal ukuran 10MB"` sebagai teks informasi saja. Tidak ada validasi programatik di `app.py` maupun di `backend/routers/documents.py`.

**Kode aktual `documents.py` (upload-temp endpoint):**
```python
contents = await file.read()
# ← Tidak ada: if len(contents) > 10 * 1024 * 1024: raise HTTPException(413)
extracted_text = ""
if filename.endswith(".pdf"):
    reader = pypdf.PdfReader(BytesIO(contents))  # ← File besar diterima langsung
```

**Re-test Status:** ❌ **FAIL — Bug Still Open**

## Recommendation

**Backend (`documents.py`):**
```python
MAX_FILE_SIZE = 10 * 1024 * 1024  # 10MB
contents = await file.read()
if len(contents) > MAX_FILE_SIZE:
    raise HTTPException(
        status_code=413,
        detail=f"File terlalu besar. Maksimum 10MB, ukuran file: {len(contents)//1024//1024}MB."
    )
```

**Frontend (`app.py`):**
```python
uploaded_file = st.file_uploader("Pilih file", type=["pdf","txt"], max_bytes=10*1024*1024)
```
