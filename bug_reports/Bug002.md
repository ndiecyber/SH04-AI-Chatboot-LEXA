## Description

`LexaChatbot.send_message()` dan `send_message_stream()` di `core/llm.py` tidak memiliki validasi input. Empty string, `None`, dan oversized input langsung diteruskan ke Groq API.

**Kode aktual v2.0 (tidak berubah):**
```python
def send_message(self, message: str) -> str:
    self.history.append({"role": "user", "content": message})
    # ← Tidak ada validasi apapun di sini
    messages_to_send = self._prepare_messages(message)
    ...
```

## Recommendation

```python
def _validate_message(self, message):
    if message is None:
        raise TypeError("Pesan harus berupa string, bukan None.")
    if not isinstance(message, str):
        raise TypeError(f"Tipe pesan tidak valid: {type(message).__name__}")
    if not message.strip():
        raise ValueError("Pesan tidak boleh kosong.")
    if len(message) > 4000:
        raise ValueError(f"Pesan terlalu panjang ({len(message)} karakter). Maks 4000.")
```
