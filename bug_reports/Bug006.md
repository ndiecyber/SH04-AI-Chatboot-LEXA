## Title
`requests` Library Tidak Ada di `requirements.txt` — Frontend Gagal Setelah Fresh Install

## Severity
🔴 **Critical**

## Priority
🔴 **P1 — Fix Immediately**

## Description

`app.py` dan `main.py` keduanya menggunakan `import requests` untuk berkomunikasi dengan backend FastAPI. Namun library `requests` tidak terdaftar di `requirements.txt`. Pada fresh install dengan `pip install -r requirements.txt`, library ini tidak akan terinstall, menyebabkan crash saat aplikasi dijalankan.

## Steps to Reproduce

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py  # CRASH
# ModuleNotFoundError: No module named 'requests'
```

## Expected Result
`streamlit run app.py` berjalan tanpa error setelah `pip install -r requirements.txt`.

## Actual Result
```
ModuleNotFoundError: No module named 'requests'
```

## Recommendation
Tambahkan `requests` ke `requirements.txt`:
```
groq
python-dotenv
streamlit
sentence-transformers
numpy
pypdf
fastapi
uvicorn[standard]
sqlalchemy
python-multipart
requests          ← TAMBAHKAN INI
```
