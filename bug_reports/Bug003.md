## Title
Hardcoded CSS Background Color Breaks Streamlit Dark Mode

## Severity
🟡 **Medium**

## Status History

| Date | Status | Actor | Notes |
|------|--------|-------|-------|
| 2025-07-01 | 🟡 Open | QA Team | Ditemukan di v1.0 |
| 2026-07-10 | 🟡 Open | QA Team | Re-check v2.0 — CSS hardcode masih ada |

## Current Status: 🟡 **OPEN — Not Fixed**

---

## Description

CSS `background-color: #f8fafc` di-hardcode via `st.markdown(unsafe_allow_html=True)` sehingga override Streamlit dark mode.

**Kode aktual v2.0 (tidak berubah dari v1.0):**
```python
st.markdown("""
    <style>
    .main {
        background-color: #f8fafc;  ← masih ada
    }
    ...
    </style>
""", unsafe_allow_html=True)
```

**Re-test Status:** ❌ **FAIL — Bug Still Open**

---

## Recommendation

Hapus baris `background-color: #f8fafc` atau gunakan `.streamlit/config.toml` untuk theming yang kompatibel dengan dark mode.
